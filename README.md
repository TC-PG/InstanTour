# InstanTour
This project is a side project using the below tech stack: 

```
1.  MongoDB
2.  Express
3.  Node.js
```

Dev Environment: Create a **.env file** in the root directory. Add below environment variables.

```
1.  GMAILPW
2.  GEOCODER_API_KEY
3.  adminCode
4.  CLOUDINARY_API_KEY
5.  CLOUDINARY_API_SECRET
```

Then,
1.  Run `npm install`
2.  Run `nodemon | node app.js`
